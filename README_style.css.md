**README.md**
===============

**File Overview**
-----------------

This single file, `style.css`, contains CSS (Cascading Style Sheets) code responsible for defining the visual appearance and layout of an HTML webpage.

**What this file does**
----------------------

The `style.css` file defines the following visual aspects of the HTML page:

- **Body Styles**: Sets the font family, text alignment, margin, and background color of the entire webpage.
- **Button Styles**: Defines the appearance and behavior of buttons on the webpage, including their padding, font size, background color, text color, border, and hover effects.

**Functions/Classes explained**
------------------------------

### Body Styles

The `.body` styles section defines the following CSS styles:

- `.font-family`: Sets the font family to Arial, with a fallback to sans-serif in case Arial is not supported.
- `.text-align`: Centers the text horizontally within the webpage.
- `.margin-top`: Adds a top margin of 100 pixels to the webpage.
- `.background-color`: Sets the background color of the webpage to a light grayish color (`#f2f2f2`).

### Button Styles

The `.button` styles section defines the following CSS styles:

- `.padding`: Adds padding of 10 pixels on the top and bottom, and 20 pixels on the left and right sides of the button.
- `.font-size`: Sets the font size of the button text to 16 pixels.
- `.background-color`: Sets the background color of the button to blue.
- `.color`: Sets the text color of the button to white.
- `.border`: Removes the button's default border.
- `.border-radius`: Sets the button's border radius to 5 pixels.
- `.cursor`: Sets the cursor shape to a pointing hand when hovering over the button.

### Button Hover Effects

The `.button:hover` styles section defines the following CSS styles:

- `.background-color`: Sets the background color of the button to a darker blue color (`darkblue`) when the user hovers over it.

**Dependencies used**
----------------------

This file uses the following dependencies:

- **CSS** (Cascading Style Sheets) to define the visual appearance of the HTML page.
- **HTML** (Hypertext Markup Language) to structure the content of the webpage.
- **JavaScript** (optional) may be used to add interactive behavior to the webpage, but it is not used in this specific file.

**Notes**
--------

- This file is a stand-alone CSS file and does not rely on any other JavaScript or CSS files.
- This file can be used as a starting point for designing basic webpage layouts and visual effects.
- You can customize the styles to fit your specific needs and preferences.
